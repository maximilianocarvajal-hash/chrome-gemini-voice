# 🎙️ SpeakFlow: AI Conversation Partner

&gt; Extensión de Chrome para conversaciones de voz bilingües en tiempo real con Gemini Live API.

## ✨ ¿Qué hace?

**SpeakFlow** te permite hablar con la inteligencia artificial de Google (Gemini) directamente desde el panel lateral de Chrome.

- 🎤 **Conversación por voz** en tiempo real.
- 🌍 **Modo bilingüe** — alterna entre dos idiomas libremente.
- 🔑 **Pool de API keys** — rotación automática si una se agota.
- 📎 **Adjunta archivos** arrastrándolos al chat.
- 💾 **Guardado local** de conversaciones.
- 🎚️ **VAD automático** — detecta cuándo has terminado de hablar.

## 🚀 Instalación

1. Descarga esta carpeta completa (o clona el repositorio).
2. Abre **Google Chrome** y ve a:
chrome://extensions/

3. Activa el **modo Desarrollador** (interruptor arriba a la derecha).
4. Haz clic en **"Cargar descomprimida"** (Load unpacked).
5. Selecciona la carpeta donde están los archivos de la extensión.
6. Haz clic en el icono de la extensión para abrir el panel lateral.

> Si actualizas los archivos, vuelve a `chrome://extensions/` y haz clic en 🔄 (recargar) en la tarjeta de la extensión.
